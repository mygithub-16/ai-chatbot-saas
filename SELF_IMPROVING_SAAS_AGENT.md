# Self-Improving SaaS Agent

The app can evolve into a self-improving system by recording outcomes and using them to improve prompts, FAQs, and follow-up workflows.

## Signals

- Questions with low confidence or repeated clarification.
- High-intent phrases that do not convert.
- Businesses with weak demo-to-lead conversion.
- Frequently requested services missing from the business profile.

## Safe Automation

Start with recommendations in the admin dashboard before allowing automatic prompt/profile updates.
